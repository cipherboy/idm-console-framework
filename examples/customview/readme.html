Custom View Plug-in Example Readme

Example

This example shows how to plug in to the Netscape Admin Server 4.0 Custom View framework. Once the plug-in is installed, the Custom View selector will contain one more option. If the user selects the SuperMail custom view, it displays a different view for the user.

Installation Steps

  1. Install the plug-in and issue this command:

    2. dapmodify -a -f customview.ldif
  2. Compile the SuperMailCustomView.java into SuperMailCustomView.class.
  3. Jar the class file into SuperMail.jar.
  4. Make an English localization jar file by copying SuperMail.jar to SuperMail_en.jar.
  5. Copy SuperMail.jar and SuperMail_en.jar to <server-root>/java/jars.
  6. Start the Console and log in.
  7. You see the SuperMail plug-in as one item in the Custom View selector.
