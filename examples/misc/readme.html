Misc Plug-in Example Readme

Example

This directory contains three example files for calling individual components in the Netscape Admin Server 4.0 SDK framework:
  • ACLExample.java
      • Example that calls the ACL editor in the Netscape Console SDK.
  • GetExample.java and PostExample.java
      • Example that uses the Netscape Console 4.0 SDK comm package to communicate using the HTTP protocol.
